# Guardians

オンラインゲーム「Destiny2」で一緒に遊べる仲間を見つける場所を提供

# ペルソナ像
性別：男性
年齢：22歳
仕事：三重にある自動車部品メーカーでライン業務を担当している（土日休み）
社歴：社会人4年目（高卒)
役職：なし（平社員）
彼女：なし
同居人数：１人暮らし

好きなことはDestiny2（ゲーム）で遊ぶこと

ゲーム歴12年目（Destinyシリーズは5年遊んでいる）
対人戦では負けたくないので一緒に遊ぶ人は強い人が望ましい
